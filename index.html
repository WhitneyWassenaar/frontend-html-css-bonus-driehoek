Hoe teken je een driehoek in CSS

  1. Maak een div met class driehoek of gebruik 'div' type selector
  2. Zet width en height op 0
  3. Maak eerst een vierkant.
    Voeg border-top, border-bottom, border-left en border-right toe.
  4. Geef bij elke border property: 200px solid (kleur naar keuze).
    Als je elke border-property een eigen kleur geeft, dan kan je goed zien welke driehoek bij welke property hoort.
  5. Maak 3 van de 4 border properties transparent
  6. Je hebt een mooie driehoek getekend.
